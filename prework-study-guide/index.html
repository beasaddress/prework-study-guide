Prework Study Guide

HTML

  • the purpose of HTML is to apply meaning, not presentation
  • the head element contains information about the webpage
  • the body element represents the visible content shown to the user
  • HTML uses TAGS to define and format content
  • HTML also uses something called attributes which provide additional information about tags

CSS

  • CSS stands for Cascading Style Sheets
  • CSS concepts and syntax are used to style and format web pages
  • CSS is used in conjunction with HTML
  • together, these two languages control the layout, fonts, colors, and other visual aspects to be appreicated by the user

Git Section

  • Git is a version control system that manages and tracks changes to code
  • it allows multiple developers to work on the same codebase in a clean, collabrative way.
  • Here are some git commands to remember:
  • git status: this will tell you what branch you are working on
  • git checkout -b branch-name: creates a new branch and switches you to it

JavaScript

  • JS is a programming language that creates dynamic and interactive web content
